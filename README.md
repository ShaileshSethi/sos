# SafeZones GNDU - Emergency App

This is a fast, offline-friendly emergency web app designed for the students and residents of GNDU, Amritsar. It provides immediate access to emergency contacts, location services, and a life-saving AI voice guide.

## Features
* **One-Tap SOS:** Immediately gets your location and provides buttons to call police and ambulance.
* **AI Voice Guide:** Speaks clear, calm instructions in English.
* **Personal Siren:** Plays a loud siren sound and flashes the screen to attract attention.
* **Live Location:** Fetches your GPS coordinates and street address.
* **Visual First Aid:** Easy-to-understand first aid guide.
* **Local Safe Zones:** A map of hospitals, police stations, and shelters near GNDU.
* **Interactive UI:** A modern, "glossy" dark mode with 3D-tilting cards and a cursor spotlight.
